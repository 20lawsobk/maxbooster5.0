# Social Autopilot - Social Media Content Management Platform

## Overview

Social Autopilot is a comprehensive social media content management platform built as a full-stack web application. This is a product of B-Lawz Music business. The system enables users to create, optimize, schedule, and analyze social media content across multiple platforms including Twitter, Instagram, LinkedIn, Facebook, and TikTok. The application features AI-powered content optimization, real-time analytics, and automated publishing capabilities with a modern, responsive user interface.

## User Preferences

Preferred communication style: Simple, everyday language.

## System Architecture

### Frontend Architecture
- **React 18** with TypeScript for type-safe component development
- **Vite** as the build tool and development server for fast hot module replacement
- **Wouter** for lightweight client-side routing instead of React Router
- **Tailwind CSS** with custom CSS variables for consistent theming and responsive design
- **shadcn/ui** component library built on Radix UI primitives for accessible, customizable components
- **TanStack React Query** for server state management, caching, and API synchronization
- **React Hook Form** with Zod validation for form handling and schema validation

### Backend Architecture
- **Express.js** server with TypeScript for API endpoints and middleware
- **RESTful API** design with structured route handlers for content, platforms, and analytics
- **Memory storage** implementation with interfaces designed for easy database migration
- **Request/response logging** middleware for development debugging
- **Error handling** middleware with standardized error responses

### Component Architecture
- **Modular component structure** with separate UI primitives and business logic components
- **Custom hooks** for mobile detection, toast notifications, and theme management
- **Context providers** for theme and form state management
- **Compound component patterns** for complex UI elements like forms and modals

### Content Management System
- **Rich text editor** with formatting tools, hashtag generation, and content statistics
- **Platform optimization** scoring system that evaluates content against platform-specific constraints
- **Scheduling system** with optimal posting time recommendations
- **Content preview** system showing how content appears on different platforms
- **Template saving** functionality for reusable content patterns

### Data Models
- **Users** with username/password authentication
- **Platforms** with connection status, character limits, and branding
- **Content** with versioning, optimization scores, and engagement predictions
- **Analytics** tracking engagement metrics across platforms

### State Management
- **React Query** for server state with optimistic updates and background refetching
- **React Context** for theme preferences and global UI state
- **Local state** with useState and useReducer for component-specific data
- **Form state** managed by React Hook Form with Zod schema validation

### Styling and Theming
- **CSS custom properties** for dynamic theming with light/dark mode support
- **Tailwind utility classes** with custom color palette and spacing system
- **Component variants** using class-variance-authority for consistent styling
- **Responsive design** with mobile-first approach and breakpoint-specific layouts

## External Dependencies

### Database and Storage
- **Drizzle ORM** configured for PostgreSQL with schema-first development approach
- **@neondatabase/serverless** for PostgreSQL connection in serverless environments
- Database migrations managed through Drizzle Kit with schema versioning

### UI and Styling Libraries
- **Radix UI** primitives for accessible, unstyled components including dialogs, dropdowns, and form controls
- **Tailwind CSS** for utility-first styling with PostCSS for processing
- **Lucide React** for consistent iconography throughout the application
- **class-variance-authority** for type-safe component variant management

### Development and Build Tools
- **TypeScript** for static type checking across frontend and backend
- **ESBuild** for fast backend bundling in production builds
- **Vite plugins** including React support and Replit-specific development tools
- **tsx** for running TypeScript files directly in development

### Form and Validation
- **React Hook Form** for performant form handling with minimal re-renders
- **Zod** for runtime type validation and schema definition
- **@hookform/resolvers** for integrating Zod validation with React Hook Form

### External Services Integration
- **Platform APIs** ready for integration with Twitter, Instagram, LinkedIn, Facebook, and TikTok
- **Analytics tracking** infrastructure for engagement metrics and performance monitoring
- **Content optimization** system designed for AI-powered suggestions and improvements
